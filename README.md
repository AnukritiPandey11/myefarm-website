# Mye Farm Website

A responsive product showcase website developed for **Mye Farm**, an organic farm-to-home startup focused on delivering fresh and natural products directly to customers.

This project was built to provide customers with an easy way to explore products, learn about the brand, and place orders through WhatsApp integration.

---

## Features

- Product catalog with images and details
- Responsive design for mobile and desktop devices
- WhatsApp ordering integration
- Delivery information section
- About Mye Farm section
- Clean and user-friendly interface

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## Project Type

Client project developed for a real business requirement.
